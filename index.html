2048 Cupcakes – Combine Delicious Tiles in this Addictive Puzzle Game

2048 is a popular single-player sliding block puzzle game created by Gabriele Cirulli. The game is played on a 4×4 grid, and the objective is to combine tiles with the same number to create a tile with the value of 2048. Each time the player makes a move, a new tile appears on the board with a value of either 2 or 4.

The player can move the tiles in four directions: up, down, left, and right. When the player makes a move, all the tiles on the board slide as far as possible in the chosen direction. If two tiles with the same value collide, they merge into a single tile with the value of their sum. The game ends when there are no more moves to make, either because the player has successfully merged all the tiles into a single tile with the value of 2048, or because there are no more moves that can be made.
